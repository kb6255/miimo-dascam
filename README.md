# USB 双摄行车记录仪 软件说明书

## 概述

USB 双摄行车记录仪是一款基于 PyQt5 开发的车载视频录制软件，支持双摄像头同时工作，具备录像、截图、视频锁定、回放等功能。软件运行在嵌入式 Linux 系统上，通过 USB 摄像头采集视频，自动保存到 TF 卡（USB 存储设备）。

---

## 系统要求

- **操作系统**：Linux（支持 X11/Wayland）
- **Python 版本**：Python 3.x
- **依赖库**：
  - OpenCV (cv2)
  - NumPy
  - PyQt5
- **硬件**：
  - 2 个 USB 摄像头（支持 V4L2）
  - USB 存储设备（TF 卡/USB 盘）
  - 显示器（推荐 1280x720 或更高分辨率）

---

### 安装依赖

## 功能操作

### 1. 录像功能

#### 开始/停止录像
- 点击 **「开始录像」** 按钮开始录制
- 按钮变为绿色 **「停止录像」**，点击可停止录制
- 录像文件自动按时间命名，如 `20260708_123456.mp4`

#### 自动分段
- 默认每 **1 分钟** 自动分段（可在设置中调整，范围 1-10 分钟）
- 分段时自动保存上一段视频
- 分段过程中不会丢失画面

#### 锁定视频
- 在录像过程中点击 **「锁定视频」** 按钮
- 状态栏显示「已标记锁定，分段结束时自动保存」
- 当前视频分段结束后，会自动复制一份到 `lock_video/` 目录
- 锁定文件名以 `LOCK_` 开头，如 `LOCK_20260708_123456.mp4`

### 2. 模式切换

- 点击 **「前进模式」** 按钮切换摄像头主次
- **前进模式**：主摄像头显示前画面，副小窗显示后画面
- **后退模式**：主摄像头显示后画面，副小窗显示前画面

### 3. 截图功能

- 点击 **「截图」** 按钮
- 当前画面保存为 JPG 图片
- 截图保存到 `photos/` 目录，文件名格式：`shot_20260708_123456.jpg`

### 4. 相册/录像回放

- 点击 **「相册/录像」** 按钮打开文件列表面板
- 切换标签页查看 **「录像文件」** 或 **「截图文件」**
- **双击文件** 可播放视频或查看图片

#### 视频播放器
- 支持播放/暂停控制
- 支持进度条拖动
- 支持连续播放模式（在设置中开启）
- 点击右下角 **×** 关闭播放器

## 设置菜单

点击左上角 ⚙ 按钮打开设置菜单，包含以下选项：

### 1. 录制每段时长
- 调整录像自动分段时间
- 范围：1-10 分钟
- 默认：1 分钟

### 2. 时间水印
- 开关控制是否在录像中显示时间水印
- 水印显示格式：`2026-07-08 12:34:56`
- 默认：开启

### 3. 视频播放模式
- **单次播放**：播放完当前视频后停止
- **连续播放**：自动连续播放列表中的所有视频
- 默认：单次播放

### 4. 存储空间

显示 TF 卡/USB 存储设备的使用情况：
- 可用空间 / 总空间
- 已用空间百分比进度条
- 存储路径

#### 清空存储空间
- 点击红色 **「清空存储空间」** 按钮
- 弹出确认对话框，提示将删除的文件类型
- **删除范围**：
  - 所有普通录像文件（`.mp4`）
  - 所有截图文件（`.jpg`, `.png`）
- **保留文件**：
  - 锁定的视频文件（`LOCK_*.mp4`）
  - 目录结构（`record/`, `lock_video/`, `photos/`）

> ⚠️ **重要提示**：清空操作不可恢复，请确认后再执行！

## 存储管理

### 自动循环存储
- 软件默认启用循环存储，最大容量 **30GB**
- 当录像文件总大小超过 30GB 时，自动删除最旧的非锁定视频
- 可在 `config.json` 中调整 `cycle_storage_gb` 参数

### 空间不足警告
- 当剩余空间低于 **5GB** 时，状态栏显示警告提示
- 可在 `config.json` 中调整 `warn_space_gb` 参数

### 定时检查
- 每 **30 秒** 自动检查磁盘空间
- 每 **5 秒** 自动刷新文件列表

## 配置文件

配置文件 `config.json` 位于程序目录：

```json
{
  "video_width": 1280,
  "video_height": 720,
  "split_minute": 1,
  "enable_watermark": true,
  "cycle_storage_gb": 30,
  "warn_space_gb": 5,
  "play_mode": "single"
}
```

### 配置项说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `video_width` | 1280 | 视频宽度（像素） |
| `video_height` | 720 | 视频高度（像素） |
| `split_minute` | 1 | 录像分段时长（分钟） |
| `enable_watermark` | true | 是否启用时间水印 |
| `cycle_storage_gb` | 30 | 循环存储最大容量（GB） |
| `warn_space_gb` | 5 | 空间不足警告阈值（GB） |
| `play_mode` | "single" | 播放模式：single/continuous |

## 常见问题

### Q: 摄像头无法识别？
A: 请检查：
1. USB 摄像头是否正确连接
2. 是否有 V4L2 驱动支持
3. 运行 `v4l2-ctl --list-devices` 查看设备列表

### Q: 录像文件无法播放？
A: 可能原因：
1. 录像过程中异常中断
2. 使用其他播放器尝试（推荐 VLC）
3. 检查文件是否完整

### Q: 存储空间满了怎么办？
A: 解决方法：
1. 手动删除不需要的视频文件
2. 使用设置中的「清空存储空间」功能
3. 调整 `cycle_storage_gb` 参数减小循环存储容量

### Q: 如何锁定重要视频？
A: 操作步骤：
1. 在录像过程中点击「锁定视频」按钮
2. 等待当前分段结束
3. 锁定的视频会自动保存到 `lock_video/` 目录
4. 锁定的视频不会被自动清理

## 部署与批量烧录

### 方案一：一键配置（最简单）

适用于单台或少量树莓派的部署：

```bash
# 1. 将项目文件复制到树莓派
scp -r /home/kongbin/usbcamra pi@raspberrypi:/home/pi/

# 2. 在树莓派上运行
cd /home/pi/usbcamra
sudo bash setup.sh
```

**配置内容：**
- ✓ 开机自启动 dashcam 服务
- ✓ 静默启动（隐藏 logo、控制台输出）
- ✓ 禁用 Plymouth 启动动画
- ✓ 自动登录 pi 用户

### 方案二：自定义镜像（批量烧录推荐）

适用于批量生产，一次构建，无限刷机：

#### 步骤 1：准备基础镜像
```bash
# 下载官方树莓派 OS 镜像
wget https://downloads.raspberrypi.com/raspios_arm64/images/raspios_arm64-2024-11-19/2024-11-19-raspios-bookworm-arm64.img.xz

# 解压
xz -d 2024-11-19-raspios-bookworm-arm64.img.xz
```

#### 步骤 2：构建自定义镜像
```bash
cd /home/kongbin/usbcamra/deploy
sudo bash create-image.sh ../2024-11-19-raspios-bookworm-arm64.img
```

#### 步骤 3：批量刷机
```bash
# 使用 dd 刷入 TF 卡
sudo dd if=dashcam-os.img of=/dev/sdX bs=4M status=progress

# 或使用 Raspberry Pi Imager
# 选择 "Use custom" → 选择 dashcam-os.img
```

#### 步骤 4：首次启动
- 插入 TF 卡到树莓派
- 通电启动
- 系统自动运行 dashcam 程序
- **无需任何配置**

### 方案三：使用烧录脚本（最省事）

```bash
cd /home/kongbin/usbcamra/deploy
sudo bash deploy.sh --app-dir /home/pi/dashcam --user pi
```

## 静默启动配置说明

以下参数在 `/boot/firmware/cmdline.txt` 中配置：

| 参数 | 作用 |
|------|------|
| `quiet` | 减少启动日志输出 |
| `splash` | 启用启动画面（被 Plymouth 控制） |
| `plymouth.ignore-serial-consoles` | 忽略串口控制台的 Plymouth |
| `logo.nologo` | 禁用树莓派彩虹 logo |
| `vt.global_cursor_default=0` | 隐藏光标 |
| `console=tty1` | 仅保留 TTY1 控制台 |

## 注意事项

1. **首次使用**：请确保 USB 存储设备已正确挂载且有写入权限
2. **正常关机**：请先停止录像再关闭软件，避免视频文件损坏
3. **存储设备**：建议使用高速 TF 卡（Class 10 或更高）
4. **温度**：避免在高温环境下长时间使用，可能影响录像质量
5. **清理**：定期清理不需要的视频文件，保持足够存储空间
